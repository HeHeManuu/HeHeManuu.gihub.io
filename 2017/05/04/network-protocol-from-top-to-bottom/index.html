网络分层及相关协议

发表于 2017-05-04 | 分类于 基础网络知识 |

OSI模型把网络通信的工作分为7层，分别是物理层、数据链路层、网络层、传输层、会话层、表示层和应用层。TCP/IP模型分为四层，分别为网络接口层，网络层，传输层，应用层。下面主要介绍TCP/IP模型中的主要协议。

TCP/IP协议一览

层次 主要协议 数据单位
应用层 Telnet,FTP,HTTP,SMTP,DNS 报文
传输层 TCP,UDP,SCTP 报文段,用户数据报,分组
网络层 IP,ICMP,ARP,IGMP 数据报
网络接口层 以太网协议,HDLC,PPP 帧,比特

应用层

应用层为操作系统或网络应用程序提供访问网络服务的接口。应用层协议的代表包括：Telnet、FTP、HTTP、SMTP、DNS等。
Telnet：是Internet远程登陆服务的标准协议和主要方式，它为用户提供了在本地计算机上完成远程主机工作的能力。常见端口：23。
FTP：用于Internet上的控制文件的双向传输。21端口用于连接，20端口用于传输数据。
